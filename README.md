 GameHub – A Game Library

An engaging **React + Firebase** web app for discovering, exploring, and supporting game developers.
Users can browse indie games, view details, and manage their profiles — all in one place.

---

 Live Demo

 **Live URL:** https://remarkable-yeot-d4e152.netlify.app/


##  Project Purpose

**GameHub** was built to provide a smooth and interactive experience for gamers and developers.
It allows users to:

* Discover top-rated games.
* Learn about game developers.
* View game details in a beautiful UI.
* Register, log in, and manage their profiles securely with Firebase Authentication.
* Reset forgotten passwords.
* Enjoy a responsive and animated interface powered by Tailwind CSS and Framer Motion.



##  Pages Overview

| Page                | Description                                                  |
| ------------------- | ------------------------------------------------------------ |
| **Home**            | Banner slider, popular games section, and newsletter signup. |
| **All Games**       | Displays all available games dynamically from JSON data.     |
| **Game Details**    | Protected page showing detailed game info.                   |
| **Login**           | Email-password and Google login options.                     |
| **Register**        | New user signup with name, email, photo, and password.       |
| **Forgot Password** | Reset password feature that redirects to Gmail.              |
| **My Profile**      | Displays user info with an option to update profile details. |
| **Update Profile**  | Form to update user name and profile picture.                |
| **Extra Info**      | Additional informational page about gaming awareness.        |
| **404 Page**        | Custom “Page Not Found” for invalid routes.                  |

##  Key Features

✅ **Responsive Layout** — Works perfectly on all devices.
✅ **Firebase Authentication** — Secure login, register, Google login, and password reset.
✅ **Protected Routes** — Restricts game details & profile pages for logged-in users only.
✅ **Dynamic Titles** — Browser tab title changes based on the current page.
✅ **Game JSON Data** — All games fetched dynamically from a local `games.json` file.
✅ **Framer Motion Animations** — Smooth transitions and hover effects.
✅ **Update Profile** — Users can update their name and photo URL.
✅ **Forgot Password Flow** — Redirects user to Gmail after reset.
✅ **Clean Component Structure** — Separated components, pages, and layouts for clarity.
